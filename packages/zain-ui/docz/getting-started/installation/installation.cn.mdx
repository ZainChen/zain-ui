# 安装

Zain-UI 作为 npm 软件包提供。

#### 使用 npm 或 yarn

我们建议使用 npm 或 yarn 进行安装，这不仅使开发更容易，而且还使您可以利用 Javascript 包和工具的丰富生态系统。

```tsx
// with npm
npm install zain-ui

// with yarn
yarn add zain-ui
```

请注意，react> = 16.8.0和react-dom> = 16.8.0是对等方依赖性。
