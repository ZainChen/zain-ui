# インストール

Zain-UI は npm パッケージ として入手可能です。

#### npm または糸の使用

インストールにはnpmまたはyarnを使用することをお勧めします。これにより、開発が容易になるだけでなく、Javascriptパッケージおよびツールの豊富なエコシステムを利用できるようになります。

```tsx
// with npm
npm install zain-ui

// with yarn
yarn add zain-ui
```

react> = 16.8.0とreact-dom> = 16.8.0はピアの依存関係であることに注意してください。
